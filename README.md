# Triple A Web

A simple website to "show of" the Triple A discord bot, developed with **Next.js** and **TailwindCSS**

## Development

### Setup
1. This app is developed with `Node 20.x`, so make sure this is installed and active before continuing.
2. Pull down the local environment variables from vercel with `vercel env pull`
3. Install all node modules with `npm i`
4. Run the development environment with `npm run dev`, and the app is available at [http://localhost:3000](http://localhost:3000)
